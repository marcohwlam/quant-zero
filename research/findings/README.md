# Findings

Research outcomes for strategies that have completed backtesting and Gate 1 evaluation.

## Naming Convention

`NN_strategy_name_vN_YYYY-MM.md` — matches hypothesis number, version, and evaluation date.

## Required Sections per Findings File

1. **Gate 1 Verdict** — PASS / FAIL / CONDITIONAL PASS (use standard format from criteria.md)
2. **Quantitative Summary** — all Gate 1 metrics
3. **Qualitative Assessment** — economic rationale validity, look-ahead bias check, overfitting risk
4. **Key Observations** — what we learned, regime dependence, parameter sensitivity notes
5. **Next Steps** — promote, refine, or archive

## Lifecycle Tags

- `PROMOTED` — passed Gate 1, moved to paper trading
- `FAILED` — did not meet Gate 1 thresholds
- `ARCHIVED` — abandoned (no path forward identified)
- `REVISION_NEEDED` — failed but salvageable with refinement
